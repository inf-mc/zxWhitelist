# A whitelist used spigot-api.
Need to connect to **MySQL** database.  
The configuration file is in **'plugins\zxWhiteilst\config.yml'**.  
## Commands
- `whitelistaddplayer 'name'`  
  **simple:`wla 'name'`**
- `whitelistdelete 'name'`  
  **simple:`wld 'name'`**
- `whitelistinqure`  
  **simple:`wli`**
